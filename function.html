Function

Introduction

functions is group of related statements, that can perform specific tasks. it can be executed, when we call that function only by its name

Advantages

  • Secure
  • code reusability
  • better understanding
  • compact

Function with Zero argument

Input Output

def new()
print("success")
new() 		success

Function with One argument

Input Output

def new(a)
print("success")
new(a) 		success

Function with Two or Multi argument

Input Output

def new(a,b)
print(a, "Scored", b, "runs")
new("Dhoni", 100) 		Dhoni scored 100 runs

Important points

  • It can perform only when given argument equal to output argument, if it less than or greater than, it will show error
    • Input Output
    def fun(a,b)
    print(a, "Scored", b, "runs")
    fun("dhoni")     		fun() missing 1 required positional argument: 'b'
    ex :2
    Input Output
    def fun(a,b)
    print(a, "Scored", b, "runs")
    fun("dhoni",100,50)     		fun() takes 2 positional arguments but 3 were given
  • It can perform when we declare the argument with input, it possible only 2nd argument
    • Input Output
    def fun(a,b=100)
    print(a, "Scored", b, "runs")
    fun("dhoni")     		Dhoni scored 100 runs
    ex :2
    Input Output
    def fun(b,a="Dhoni")
    print(a, "Scored", b, "runs")
    fun("100,)     		Dhoni scored 100 runs
  • Code Reusability
    • Input Output
    def fun(a,b):
    print(a+b)
    fun(10,20)
    fun(200,765)     		30
    965
  • We can declare multiple inputs for sigle argument by using * in fornt of argument, output will came in Tuples
    • Input Output
    def fun(*a):
    print(a)
    fun(10, "dhoni")
    fun(10,20)
    fun(10,20,30,40,50)     		(10, 'dhoni')
    (10, 20)
    (10, 20, 30, 40, 50)
  • We can declare multiple arguments as input by using ** in fornt of argument, output will came in Dictionary
    • Input Output
    def fun(**a):
    print(a)
    fun(10, "dhoni")
    fun(a=10)
    fun(b=10,c=20)     		{'a': 10}
    {'b': 10, 'c': 20}

    def fun(a,b):
