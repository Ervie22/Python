Comprehension

Introduction

List comprehension offers a shorter syntax when you want to create a new list based on the values of an existing list.

Ex:1 Basic Print statement

Traditional Comprehension output
var=[5,10,2,3]
x=[]
for a in var:
x.append(a)
print(x) 		var=[5,10,2,3]
z=[x for x in var]
print(z) 		[5, 10, 2, 3]
