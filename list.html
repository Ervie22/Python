    Important Keys

    • var.append(), used to add single element in last in list
      Input Output
      var=["dhoni","azar","balu","kholi"]
      var.append("jadeja")
      print(var)       		['dhoni', 'azar', 'balu', 'kholi', 'jadeja']
    • var.extend(), used to add multiple element in last in list
      Input Output
      var=["dhoni","azar","balu","kholi"]
      var.extend(["jadeja","rahul"])
      print(var)       		['dhoni', 'azar', 'balu', 'kholi', 'jadeja', 'rahul']
    • var.pop(), if empty remove last element in list, if give an particular index it will remove
      Input Output
      var=["dhoni","azar","balu","kholi"]
      var.pop()
      print(var)       		['dhoni', 'azar', 'balu']

      Input Output
      var=["dhoni","azar","balu","kholi"]
      var.pop(2)
      print(var)       		['dhoni', 'azar', 'kholi']
    • var.remove(), remove a particular string
      Input Output
      var=["dhoni","azar","balu","kholi"]
      var.remove("azhar")
      print(var)       		['dhoni', 'balu', 'kholi']
    • var.insert(), insert a string in particular index in list
      Input Output
      var=["dhoni","azar","balu","kholi"]
      var.insert(1,"jadeja")
      print(var)       		['dhoni', 'jadeja', 'azar', 'balu', 'kholi']
    • var.sort(), sort in ascending order
      Input Output
      var=["dhoni","azar","balu","kholi"]
      var.sort()
      print(var)       		['azar', 'balu', 'dhoni', 'kholi']
    • var.sort(reverse.True), sort in descending order
      Input Output
      var=["dhoni","azar","balu","kholi"]
      var.sort(reverse=True)
      print(var)       		['kholi', 'dhoni', 'balu', 'azar']
    • len(var), to find length of list
      Input Output
      var=["dhoni","azar","balu","kholi"]
      print(len(var))       		4
    • var.clear(), clear list
      Input Output
      var=["dhoni","azar","balu","kholi"]
      var.clear()
      print(var)       		[]
    • del(var), delete main memory
      Input Output
      var=["dhoni","azar","balu","kholi"]
      var.delete()
      print(var)       		NameError: name 'var' is not defined. Did you mean: 'vars'?
