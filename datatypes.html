Slicing

Extracting of a particular word or portion in a string

Can perform in 6 ways

  1. using index (if a="vijay", v starts from 0, in value V starts from 1)slice j (a[2])
  2. Using Negative Value (if a="vijay", y starts from -1)(a[-3])
  3. Using index & Value (a[2:3])
  4. Using Negative Value & Negative Value (a[-3:-2])
  5. Using index & Negative Value (a[2:-2])
  6. Using Negative Value & index (a[-3:3])

SET

Introduction

  • Collection of unordered elements enclosed with curly brasis seperated by comma called list , ex: a={"vijay",7}
  • Its not ann array, can perform different types
  • Slicing cannot perform, Due to unordered list
  • Mutable
  • Duplicates not allowed

    • Find data type

    a={"dhoni", "kholi", 7, 8}, will find data type by follwing code

    Input Output
    a={"dhoni", "kholi", 7, 8}}
    print(a)
    print(type(a))     		{"kholi","dhoni", 8, 7}
    class 'Set'

    Important Keys

      Input Output

    • var.remove(), used to remove a particular element in set
      Input Output
      var={"dhoni", "azhar", "yuvi", 7, 10}

      print(var)       		{'yuvi', 'dhoni', 7, 10}
    • var.clear(), used to clear set
      Input Output

    • len(var), to find length of set
