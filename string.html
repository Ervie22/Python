Slicing

Extracting of a particular word or porting in a string

Can perform in 6 ways

  1. using index (if a="vijay", v starts from 0, in value V starts from 1)slice j (a[2])
  2. Using Negative Value (if a="vijay", y starts from -1)(a[-3])
  3. Using index & Value (a[2:3])
  4. Using Negative Value & Negative Value (a[-3:-2])
  5. Using index & Negative Value (a[2:-2])
  6. Using Negative Value & index (a[-3:3])
