SET

Introduction

  • Collection of unordered elements enclosed with curly brasis seperated by comma called list , ex: a={"vijay",7}
  • Its not ann array, can perform different types
  • Slicing cannot perform, Due to unordered list
  • Mutable
  • Duplicates not allowed

    • Find data type

    a={"dhoni", "kholi", 7, 8}, will find data type by follwing code

    Input Output
    a={"dhoni", "kholi", 7, 8}}
    print(a)
    print(type(a))     		{"kholi","dhoni", 8, 7}
    class 'Set'

    Important Keys

    • var.add(), used to add an element in set
      Input Output
      var={"dhoni", "azhar", "yuvi", 7, 10}
      var.add("sachin")
      print(var)       		{'azhar', 'sachin', 'dhoni', 7, 10, 'yuvi'}
    • var.pop(), used to remove an element in set
      Input Output
      var={"dhoni", "azhar", "yuvi", 7, 10}
      var.pop()
      print(var)       		{7, 'azhar', 10, 'yuvi'}
    • var.remove(), used to remove a particular element in set
      Input Output
      var={"dhoni", "azhar", "yuvi", 7, 10}
      var.remove("azhar")
      print(var)       		{'yuvi', 'dhoni', 7, 10}
    • var.clear(), used to clear set
      Input Output
      var={"dhoni", "azhar", "yuvi", 7, 10}
      var.clear()
      print(var)       		set()
    • len(var), to find length of set
      Input Output
      var={"dhoni", "azhar", "yuvi", 7, 10}
      print(len(var)))       		5
    • del(var), to delete entire var
      Input Output
      var={"dhoni", "azhar", "yuvi", 7, 10}
      del(var)
      print(var)       		NameError: name 'var' is not defined. Did you mean: 'vars'?
